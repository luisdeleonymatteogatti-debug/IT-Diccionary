Webpage

A webpage is a single HTML document that is part of a website; it can include text, images, links, and embedded media and is accessed via a unique URL.

In Spanish: Página web

Website

A website is a collection of related web pages, multimedia content, and files that are hosted on a web server and accessed publicly via the internet.

In Spanish: Sitio web

Worksheet

A worksheet is a document used in spreadsheets to organize and analyze data, typically consisting of rows and columns.

In Spanish: Hoja de cálculo

Write

Writing is the process of creating text content for various purposes, such as communication, documentation, or creative expression.

In Spanish: Escribir

Web design

Web design is the process of creating websites and web pages, including layout, color scheme, and user experience considerations.

In Spanish: Diseño web

Web development

Web development is the process of creating websites and web applications, involving both frontend and backend development.

In Spanish: Desarrollo web
